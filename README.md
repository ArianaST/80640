## ¿Qué practico aquí?
Este repositorio agrupa prácticas enfocadas en:

- **Manipulación del DOM** (eventos, inputs, actualización de UI)
- **Lógica de programación** (condiciones, validaciones, operaciones)
- **Estructura semántica** con HTML
- **Estilos con CSS** (layout, tipografías, componentes simples)
- Organización básica de archivos (HTML / CSS / JS)
